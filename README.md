## Description
Next.js starter with TypeScript, Jest, Cypress, Eslint for e-commerce. <br/>
Stack: Nest, Sequelize, PostgresSQL, Swagger. <br/>

**Features:**
- API documentation
- Validation body
- CRUD for categories/products
- Authorization for admin

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
